[[v19-125-0]]
=== Salmos, 125
// naa ?, NAA - Nova Almeida Atualizada

[[v19-125-1]]

^<<v19-125-1,1>>^ Os que confiam no Senhor são como o monte Sião, que não se abala, mas continua firme para sempre.

[[v19-125-2]]

^<<v19-125-2,2>>^ Como em volta de Jerusalém estão os montes, assim o Senhor  está ao redor do seu povo, desde agora e para sempre.

[[v19-125-3]]

^<<v19-125-3,3>>^ O cetro dos ímpios não permanecerá na terra dos justos, para que os justos não comecem a praticar a iniquidade.

[[v19-125-4]]

^<<v19-125-4,4>>^ Faze o bem, Senhor , aos bons e aos retos de coração.

[[v19-125-5]]

^<<v19-125-5,5>>^ Quanto aos que se desviam para caminhos tortuosos, o Senhor  os levará juntamente com os malfeitores. Paz sobre Israel!
